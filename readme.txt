A conditional plugin that determines if the current page is a subpage of the passed page ID.

== Description ==

Is Subpage Of is a simple Wordpress conditional plugin that determines if the executing page is a subpage of the given page ID. Helpful to show content on certain document trees only.

[See plugin homepage for more details](http://www.schloebe.de/wordpress/is-subpage-of-plugin/)

== Installation ==

1. Download the plugin and unzip it.
1. Upload the folder is-subpage-of/ to your /wp-content/plugins/ folder.
1. Activate the plugin from your Wordpress admin panel.
1. Installation finished.
